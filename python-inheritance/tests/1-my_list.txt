=================================
How to Use 1-my_list.py
=================================

This module defines a class ``MyList`` that inherits from ``list``.

First, import ``MyList``:

    >>> MyList = __import__('1-my_list').MyList

Check instantiation and inheritance:
    >>> my_list = MyList()
    >>> isinstance(my_list, list)
    True
    >>> issubclass(MyList, list)
    True

Check empty list and __str__:
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

Check append() and print_sorted() with not sorted append:
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

Check that the original list remains unmodified:
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Check print_sorted() with already sorted append:
    >>> m2 = MyList()
    >>> m2.append(1)
    >>> m2.append(2)
    >>> m2.append(3)
    >>> m2.print_sorted()
    [1, 2, 3]

Check print_sorted() with negative numbers:
    >>> my_list_2 = MyList()
    >>> my_list_2.append(-5)
    >>> my_list_2.append(-1)
    >>> my_list_2.append(-10)
    >>> my_list_2.print_sorted()
    [-10, -5, -1]
