=================================
How to Use 1-my_list.py
=================================

This module defines a class ``MyList`` that inherits from ``list``.

First, import ``MyList``:

    >>> MyList = __import__('1-my_list').MyList

Now you can create a ``MyList`` instance:

    >>> my_list = MyList()
    >>> my_list
    []

You can append elements to it:

    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> my_list
    [1, 4, 2, 3, 5]

The ``print_sorted()`` method prints the list in ascending order:

    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

But the original list remains unmodified:

    >>> my_list
    [1, 4, 2, 3, 5]

It works with negative numbers too:

    >>> my_list_2 = MyList()
    >>> my_list_2.append(-5)
    >>> my_list_2.append(-1)
    >>> my_list_2.append(-10)
    >>> my_list_2.print_sorted()
    [-10, -5, -1]
